微積分甲（一）

  • 老師： 余啟哲
  • 學分： 4
  • 課程類別： 必修
  • 授課方式： 板書
  • 評分方式：
    • 平時考成績：70%，最後會再加上5%作為調分
    • 平時考成績計算 ＝（實得分數）／（小考滿分總分）＊100
    • 會考成績：30%
  • 心得：
    • 榮登修課成績最爛的科目，大一上完全不知道在幹嘛，然後就混過頭了
      因為疫情的關係，一開始上課是教授親自錄影再丟到e3上讓我們自己看，但由於拍攝剪輯都是教授一人包辦的關係，所以上片速度有點慢， 於是我一開始就去看莊重，結果學得超爛，基礎都沒打好
    • 教授上課快狠準，重點解釋完後就會挑幾題課本例題來解，希望我們能理解基本觀念然後去做習題， 所以小考基本上都考考古加上加分題（會超過滿分），如果觀念懂、習題有做應該滿簡單的，但我大一上還太菜不懂怎麼準備，所以小考都考滿爛的。 另外小考歷年來會考3-5次，我們這屆因為疫情所以只考3次
    • 後來意識到成績要去世了，開始瘋狂翻原文書 + 寫習題 + 刷考古來準備大會考，自己是把課本藍色底的重點都看過，再看例題學怎麼解 ，看完去寫微積分小組列的習體（題型都不同），如果都會基本上那一章就等於學會了
      最後大會考成績76，比我三次小考的成績都還高
  • 結論：
      教授上課觀念都教的滿仔細的，很適合想認真學微積分把基礎打好的人，小考好好準備分數都會不錯
  • 最後分數： B-

物理（一）

  • 老師： 李威儀
  • 學分： 4
  • 課程類別： 必修
  • 授課方式： 非同步線上影音
  • 評分方式：
    • 不調分
    • 小考：30%，共五次，四次滿分5分，一次滿分10分，每一分都攸關你的總成績
    • 期末：35%
    • 期中：35%
  • 心得：
    • 考古學家
    • 平時上課是到ewant平台觀看錄好的影片，所以不用實體上課還能自己分配時間看，超涼
      影片都是教基本觀念，所以不會有很複雜很難的計算，有的話也不會考
    • 每一章都會出作業，不算分會附上解答，記得寫因為小考通常會考期中一題簡單的
    • 小考：通常會有兩題，限時作答，題目通常一題作業、一題講義內容。 準備方式不要學我真的把整個講義翻過去猜會考甚麼，極度建議上網把過去所有年份的考古載下來看，題目通常都那幾題在換
      期中期末：題目都考古或作業內容，不難。助教人很好，有時候寫不清楚被扣分，他會問你觀念，如果講得出來會幫你把分數加回去
    • 考古哪裡找：交大資工考古網（因為這門課通常是資工系在修，電機系也可以修但不好抽，我也不知道我怎麼進來的）
  • 結論：
      超級涼，甜不甜因人而異，上完感覺沒學到甚麼（但物理大部分也用不太到就是了）
  • 最後分數： A

計算機概論與程式設計

  • 老師： 蔡中庸
  • 學分： 3
  • 課程類別： 必修
  • 授課方式： 投影片 + 板書 + 上機
  • 評分方式：
    • 配分通常是個謎
    • 作業 + 上機：10次
    • 期中期末筆試 + 上機考
  • 心得：
    • 這門課是由學校分發，所以分到哪個老師是隨機的，而我被分到蔡媽（老師的綽號）的班
      學的是C語言，老師會把我們當完全不會寫程式的小白來教，所以教得很慢但很仔細。
      而我剛好沒寫過程式，所以很感謝一開始有蔡媽這樣的老師幫我把基礎打好
    • 作業：每周都會出新的，5-8題，作為下周上機的基本題（會小改），所以記得要寫
      上機：3題加分題，其餘都是作業題（作業題基本上要拿滿，加分題看時間看能力，可以就多寫）
    • 期中期末筆試：考講義內容，還會手寫code，要看熟
      期中期末上機考：平時上機時的題目，有的是加分題，有的是基本題，因此每次上機後要自己把加分題補完才會寫
      沒記錯前4題不分順序一題15分，後4題一題10分， 所以要拿高分手速要快且題目要很熟
  • 結論：
      平時上機基本題都有寫出來，加分題通常只寫1題或沒寫，上機考一次60一次70，筆試一次70幾一次80幾
      每周都有功課所以偏硬，難度普通，適合沒寫過程式或想穩穩過的人
  • 最後分數： A-

邏輯設計

  • 老師： 廖育德
  • 學分： 3
  • 課程類別： 必修
  • 授課方式： 實體/線上 + 投影片
  • 評分方式：
    • 作業：20%，共7份
    • 期中：25%，兩次
    • 期末大會考：30%
    • 加分作業：寫Verilog，直接加總分
  • 心得：
    • 平時上課有兩節會是實體，剩下那一節會是線上上課
      教學內容是進位、補數、邏輯閘、FSM、Kmap等等，但目前快忘光了
    • 老師教的中規中矩，自己功課都有按時交，兩次期中難度普通，看個考古、作業題應該就會寫，但期末大會考大暴死，直接下去
      整體上對本課沒甚麼印象，可能我都沒認真在上課，目前修課只用過FSM和邏輯敘述化簡
    • Vrilog加分作業：我記得有2次還3次，一次一分，但我就爛都沒寫
  • 結論：
      七次作業平均87.4，期中兩次分別85、79，期末47，能拿B+應該是被調爛了
  • 最後分數： B+

全球工程領袖培育及探索系列

  • 老師： 田仲豪、邱一
  • 學分： 2
  • 課程類別： 選修/基本素養
  • 授課方式： 投影片
  • 評分方式：
    • 出席：20%
    • 作業：30%
    • 上課互動：50%
  • 心得：
    • 學校寄信來就報名了，後來聽學長說這堂課很硬，但還是修完了。人數關係分成不同班，帶課老師不同，剛好我這班的老師還不錯，其他班就不得而知了
    • 課程分上下部分，上半部是邱一教授帶的，會先讓我們分組，一組三個人，我們這組多收了一位
      上課就介紹一些光電或半導體產業的東西，然後老師會發問，可以舉手發表意見，趁大一大家都還不認識多舉手發言，會算互動成績，我每堂課都舉手舉爆
      下課前會用一張紙手寫回答問題，問題跟上課內容有關，認真寫個幾句，算出席成績也算作業成績
    • 每周會分配兩個主題，一個主題會抽一組報告，每組都要事先準備（自己組分到甚麼主題會先知道，不用擔心），而且規定每個人都要報告一次，所以逃不掉
      我們組是每周找一天在交誼廳討論，把重點釐清、然後大概準備個6頁的PPT
      到期中換老師前會有個期中報告，整組一起，到時候就上去講台輪流報告。我記得還會請同學評分，請趁期初拉好關係
      簡而言之就是一堆報告，所以被說很硬
    • 下半部分由田仲豪教授上課，一開始會要我們準備自我介紹，之後上課模式差不多，只差在沒有下課前的手寫問題也不用瘋狂舉手回答
    • 還有沒有報告？當然有，一周一次，但每組只要上去一次就好，報告順序抽籤決定、內容自己選，報告完就沒事了，可以去教室坐著當坨會呼吸的肉
  • 結論：
      找到好隊友，這堂課會過得很輕鬆，然後瘋狂舉手就對了
  • 最後分數： A+

大一英文（一）

  • 老師： 吳舜華
  • 學分： 2
  • 課程類別： 外語
  • 授課方式： 投影片
  • 評分方式：
    • 出席：10%
    • 課堂參與：15%
    • 作文：5%
    • 英文護照 + GEPT模擬：15%
    • 小考：5%
    • 口說：10%
    • 期中期末：共40%
    • 拍攝Vlog：10%
  • 心得：
    • 111年開始就沒有大一英文了，不過還是寫一下
    • 照課本上，老師每堂課都會問簡單的問題並點人回答，用英文回答，會做為出席率標準，因為每個人都會點到
      偶爾有分組，一起寫個學習單這樣
    • 小考忘記考甚麼了。期中期末選擇題、閱讀題那些，難度簡單。口說要跟老師1對1發表你對看完指定課本文章的心得
    • 這堂課最麻煩的就是有一堆零零總總的東西要弄，其中GEPT、英文護照跟Vlog最麻煩
      GEPT要寫四回，第一次就通過標準才會有滿分，而且寫一回就要花1個小時以上
      英文護照要集8點，老師課堂上會貢獻2點，剩下的自己去參加圓桌或是看影片寫心得拿，拿超過8點最多可以加2分
      Vlog是要我們拍攝影片，用英文介紹一個景點，我後來決定介紹新竹動物園，但整部影片不知道在拍甚麼，風切聲還比我聲音還大，可以想像老師當時看到一定滿頭問號
      作文隨時都能教，寫完交給老師就好
  • 結論：
      事情有點多，但反正之後可能不會再有這門課了，看看就好
  • 最後分數： A

新竹六燃地方學：大學社會責任與在地實踐

  • 老師： 賴雯淑
  • 學分： 2
  • 課程類別： 領域課程
  • 授課方式： 線上聽演講
  • 評分方式：
    • 出席：20%
    • 參訪：20%
    • 期中報告：20%
    • 期末報告：40%
  • 心得：
    • 每周上課會請不同的演講者來介紹新竹周邊的歷史或是特色，進會議後要留言點名算出席
    • 因為自己有參加系隊，加上這堂課是在周三晚上，所以除了一開始沒比賽時有認真作筆記，其餘的課都是點名完就掛著
    • 期初會有參訪，會到六燃工廠參觀，位置在清夜再過去一點的路上，所以離學校滿近的
      因為當時工廠內部還在整修，所以只有先看影音的導覽，再到隔壁參觀一些改建或再利用的設備，比如雞舍、創意廚房等等
      可以順便拍個照紀錄一下過程，期中報告根據這次參訪就能寫個1-2000字的心得
      期末我一樣是寫報告，規定要2000字，不想寫報告也能改用藝術創作的方式繳交，算滿自由的
  • 結論：
      又涼又甜，期中95、期末82，最後88.5，現在寫心得才發現我可能有點名沒被點到QQ，不然應該有A+
      邀請來的講者都非常認真地準備了1個半小時的演講內容，不會無聊，很推薦想認識新竹當地歷史的人來修，但聽說現在不好選了
  • 最後分數： A

電腦輔助繪圖-快速設計提案

  • 老師： 郭明浩
  • 學分： 3
  • 課程類別： 跨域選修
  • 授課方式： 實體實作
  • 評分方式：
    • 平時成績：30%
    • 期中：30%
    • 期末：40%
  • 心得：
    • 這堂課主要是用Creo來建3D模型，跟產品建模比較有關，不是做人物或場景那類的，要修之前可以考慮一下
    • 前面三堂課為了載程式弄了很久，之後才會開始教如何操作，最後會教一點渲染，才不會整個作品都是灰的。阿因為程式比較舊，所以我覺得不管是介面還是功能上都要花滿多時間去熟悉
      另外，從第一堂課開始，老師每堂課都會勸退我們，因為學這個挫折感真得超大，尤其是安裝主要程式的過程
    • 期中考：每個人要建兩個之前上課教過的建模，看老師怎麼指派你就做哪兩個，越快做完分數越高，所以有可能會被分到比較難的就做比較慢，導致成績比較低，但也沒多低就是了（沒記錯是80分起跳）
      可能輪到我的時候老師心情特別好，直接叫我做最簡單的那兩個，所以我滿快就做出來了
    • 期末提案：期末老師要大家自己想兩個可以用的物品，並用Creo建模，我自己是建了一個奇怪的筆筒跟蘋果造型的小夜燈，實際能不能用我不知道
  • 結論：
      聽說老師分數不好拿而且滿硬的，但我覺得還好loading沒有很重（我自己完全沒學過建模）
      老師人很好，上課如果有操作上聽不懂或不會做，下課都可以去問老師，他會再教你做一遍
  • 最後分數： A+

視覺藝術概論

  • 老師： 林愛真
  • 學分： 2
  • 課程類別： 跨域選修/領域課程
  • 授課方式： 投影片
  • 評分方式：
    • 期初問卷
    • 出缺席
    • 一對一線上面談
    • 期中期末提案
    • 期中期末意見發表
    • 期中期末提案
    • 期中期末報告
  • 心得：
    • 老師開的課主要教的內容都差不多，一開始會有一些創作基礎介紹，越往期末會開始介紹一些現代主義、資本主義等內容。老師也很開放，知道每個人觀點不一樣所以不會強迫大家接受她的想法
    • 老師以前是在芝加哥藝術學院任教，多把握機會跟老師一對一面談，可以討論對於期中期末的想法，也可以聊一些關於自己創作上的問題， 通常老師在聽完你的想法後，就能很快點出你的問題或是提出一些建議，所以對於平時有在創作的人來說，跟老師面談往往能學到比課程所教的還多
    • 期中需要製作關於視覺文化現象的主題研究，可以透過PPT報告也能製作影片介紹。在報告前還有一周會拿來提案，大致上就是和同學老師分享你要做什模、如何規劃等等，讓自己在之後的製作上比較有方向
    • 期末會延續期中的主題，但形式上不用再以報告為主，可以改用創作的方式，我自己是畫了幾幅水彩，透過電子書的方式串連在一起
    • 期中期末每個同學報告完後要發表意見，一次上課至少兩次，之後老師也會提供自己的想法做總結。另外，期中期末報告會比較久，所以會拖到下課時間，後面如果還有課要提早發表意見，免得意見發表的分數拿不滿
    • 老師給分也很大方，像是1對1面談，期初問卷都是填了就給滿，會有分數差異的只有期中期末的提案跟發表， 老師會看你努力程度來評分，所以即使在創作上不是那麼在行也能拿高分
  • 結論：
      分數甜，有無創作經驗的人都可以來修，但如果有的話更好，負擔上會比較小
  • 最後分數： A+
